This is an interactive 3D portfolio website that showcases my skills, experience, and projects as an Embedded Systems Engineer. Instead of a traditional resume, recruiters can drive a car through a virtual world representing my professional journey.

## ‚ú® Features

### Core Features
- **Realistic 3D Environment** - Built with Three.js, featuring day/night cycle and dynamic lighting
- **Drivable Vehicle** - Physics-based car with realistic handling (WASD controls)
- **Interactive Buildings** - Drive close to buildings to view resume information
- **Voice Control** - Navigate using voice commands (Chrome/Edge only)
- **Self-Guided Tour** - Automatic tour of all zones
- **Multiplayer** - See other visitors in real-time (optional)

### Gamification
- **Quest System** - Complete challenges to unlock achievements
- **Achievement Badges** - Collect badges for exploring all zones
- **Collectibles** - Find microcontroller models scattered around the world

### Technical Showcase
- **Embedded Hardware Models** - 3D models of microcontrollers (ESP32, STM32, Raspberry Pi)
- **Live Dashboard** - Speed, battery, position, time
- **Terminal Console** - Logs events and interactions
- **PWA Support** - Installable on mobile/desktop, works offline

### Accessibility
- **Mobile Controls** - Touch controls for smartphones
- **Multiple Camera Views** - Switch between follow, first-person, top, and cinematic views
- **Radio System** - Background music while exploring
